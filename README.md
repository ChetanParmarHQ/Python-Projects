# Python-Projects

A collection of Python projects focused on data analysis, cleaning, and visualization. Each project takes a real world dataset, explores it with Python, and turns it into clear, decision-ready insights. Every project lives in its own folder with its notebook, dataset, supporting documents, and a dedicated README.

Maintained by **Chetan Parmar**.

---

## About this repository

This repository is a growing portfolio of hands-on Python work. Every project follows the same idea: load a dataset, clean and reshape it, answer practical business questions, and visualise the results with charts.

New projects will be added over time, each as a self-contained folder.

## How to use these projects

1. Open the folder for the project you are interested in.
2. Read that project's README for context, questions, and findings.
3. Install the required libraries (listed in each project README).
4. Open the `.ipynb` notebook in Jupyter Notebook or Google Colab.
5. Run the cells top to bottom. Each notebook downloads its dataset automatically if the file is not present.
